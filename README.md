# timbot
A Slack "bot" to replace our dear friend Tim

## How to use 
- `pip install slackclient`
- set enviorment variables for `SLACK_API_TOKEN`, `SLACK_CHANNEL_NAME`, `SLACK_TIMBOT_USER_ID`
- run `python timbot.py`
